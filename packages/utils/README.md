# Collection of useful utilities

## Installation

```bash
yarn add @convidera-team/utils
```
or
```bash
npm install @convidera-team/utils --save
```

## Usage example:

```javascript
import { 
  isString,
} from '@convidera-team/utils';

if (isString('test')) {
  console.log('It is a string');
}
```

## List of functions:

```
isBoolean
isFunction
isNumber
isString
isObject
```
